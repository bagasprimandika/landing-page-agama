# Website Pendidikan Agama

Website landing page sederhana yang dibuat menggunakan HTML dan TailwindCSS untuk memenuhi tugas mata kuliah.

## 📌 Deskripsi
Website ini bertujuan untuk memberikan gambaran tentang platform pembelajaran agama yang modern dan interaktif.  
Desain dibuat clean, minimalis, dan responsif agar nyaman digunakan di berbagai perangkat.

## 🚀 Fitur
- Hero Section dengan Call To Action
- Informasi Tentang Website
- Fitur Unggulan
- Animasi Scroll (AOS)
- Responsive Design

## 🛠 Teknologi yang Digunakan
- HTML5
- TailwindCSS (CDN)
- AOS Animation Library
